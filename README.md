## Project Description

"Mesto" is a Russian word that means "a place".<br>

It is a project for the Yandex Practicum Web development course, where I created an interactive app for sharing and liking travelling photos.<br>

This project is written using vanilla JS, and it allowed me to dive deeper into OOP methodology.<br>

### Mobile Support

The landing is adapted for devices of all sizes.<br>

## How to use

Clone down this repository. You will need **npm** installed globally on your machine.<br>

```
# to clone down the repository
$ git clone https://github.com/lipnitskaite/mesto.git

# to open the target directory
$ cd mesto
```

### Installation

```
# to install dependencies
$ npm install

# to start server at localhost:8080
$ npm run dev

# to build app
$ npm run build
```
